Giggly Progress Bar Component

The Giggly Progress Bar component is designed to visually represent task progress in a playful yet professional manner. It supports both linear and circular progress indicators to accommodate different use cases, ensuring a delightful user experience.

How to Use

              
                <giggly-progress-bar [value]="50" type="linear"></giggly-progress-bar>

                <h3>Circular Progress</h3>
                <giggly-progress-bar [value]="75" type="circular" [diameter]="120"></giggly-progress-bar>
            
                <giggly-button (click)="increaseProgress()" variant="primary">Increase Progress</giggly-button>
