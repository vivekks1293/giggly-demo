Giggly Alert component

The Giggly Alert component provides an easy-to-use alert system for displaying important messages, such as success, warnings, errors, and informational messages. It enhances user experience by providing visually appealing notifications with dismiss functionality.

How to Use

              
                <giggly-button (click)="showSuccess()" variant="secondary">Show Success Alert</giggly-button>
                <giggly-button (click)="showWarning()" variant="secondary">Show Warning Alert</giggly-button>
                <giggly-button (click)="showError()" variant="secondary">Show Error Alert</giggly-button>
                <giggly-button (click)="showInfo()" variant="secondary">Show Info Alert</giggly-button>
