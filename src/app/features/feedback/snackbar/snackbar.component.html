Giggly Button Component

The Giggly Snackbar component is a lightweight notification tool designed to provide brief messages with optional actions. It offers an engaging, user-friendly way to convey information without disrupting the user's experience.

Show Snackbar

How to Use

              
                <giggly-snackbar #snackbar message="Action completed successfully!" actionText="Undo"
                    [duration]="5000" [action]="undoAction">
                </giggly-snackbar>
        
                <giggly-button (click)="showSnackbar(snackbar)" variant="primary">Show Snackbar</giggly-button>
