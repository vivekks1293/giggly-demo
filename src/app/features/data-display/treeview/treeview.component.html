Giggly Treeview Component

The Giggly Tree View component brings an interactive, expandable tree structure to your Angular applications. Designed for hierarchical data, it allows users to navigate through nested categories, folders, or menus effortlessly while maintaining a smooth and giggly UI feel.

How to Use

              
                <giggly-tree-view [treeData]="treeData" (nodeSelect)="handleNodeSelect($event)"></giggly-tree-view>
