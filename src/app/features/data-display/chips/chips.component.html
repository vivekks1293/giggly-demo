Giggly Chip Component

The Giggly Chips component offers a stylish way to display tags, labels, or selected values in a compact and interactive form. Designed to reflect the playful and clean aesthetic of the Giggly UI system, these chips add both functionality and charm to your app's interface.

Chips Preview

How to Use

              
                <giggly-chips *ngFor="let chip of chips; let i = index" [label]="chip" [removable]="true"  
                color="secondary" (removed)="removeChip(i)"></giggly-chips>
 
 <giggly-chips label="Success Chip" color="success"></giggly-chips>
 
 <giggly-chips label="Danger Chip" color="danger" [removable]="true" 
                (removed)="alert('Removed!')"></giggly-chips>
