Giggly Badge Component

The Giggly Button component provides a set of reusable buttons with various variants and states. These buttons are designed to be playful, interactive, and user-friendly.

          <giggly-button [variant]="'primary'" [disabled]="true">Disabled Button</giggly-button>
          <giggly-button variant="playful">Playful Button</giggly-button>
