Giggly Button Component

he Giggly Sidebar/Drawer is a flexible and stylish component designed to provide collapsible side navigation for web applications. It enhances user experience by offering an organized way to access menu items and features while maintaining a playful and professional Giggly theme.

How to Use

              
                <giggly-sidebar [isOpen]="true" position="left" (toggle)="handleSidebarToggle($event)">
                <ul>
                  <li>Dashboard</li>
                  <li>Analytics</li>
                  <li>Reports</li>
                  <li>Settings</li>
                </ul>
              </giggly-sidebar>
