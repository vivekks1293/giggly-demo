Giggly Modal Component

The Giggly Modal component is an essential UI element for displaying pop-up content in a structured and visually appealing manner. It provides a flexible layout with distinct sections for the header, content, and footer, allowing developers to customize it according to their needs.

How to Use

              
                <giggly-card header="Scrollable Card" footer="Footer Text" width="300px" height="200px" [scrollable]="true" style="display: flex; flex-direction: column;">
                <div style="flex: 1; overflow-y: auto;">
                  <p>
                    This is a long content inside the card that might require scrolling. You can control the card's height 
                    and width dynamically from the application consuming it.
                  </p>
                  <p>Additional content to demonstrate scrolling behavior.</p>
                </div>
                <div style="position: sticky; bottom: 0; background: #fff; padding: 10px; border-top: 1px solid #ccc;">
                  Footer Text
                </div>
              </giggly-card>
