Giggly Tabs Component

The Giggly Tabs component is a versatile and user-friendly UI element designed to organize content into multiple sections, allowing users to switch between them effortlessly. With a declarative structure and responsive design, it ensures a seamless experience across various screen sizes.

How to Use

              
                <giggly-tabs [activeTabIndex]="0" style="width: 100%;">
                <giggly-tab name="Tab 1">
                  <!-- <giggly-tab-content> -->
                  <p>Content for Tab 1</p>
                  <!-- </giggly-tab-content> -->
                </giggly-tab>
          
                <giggly-tab name="Tab 2">
                  <!-- <giggly-tab-content> -->
                  <p>Content for Tab 2</p>
                  <!-- </giggly-tab-content> -->
                </giggly-tab>
          
                <giggly-tab name="Tab 3">
                  <!-- <giggly-tab-content> -->
                  <p>Content for Tab 3</p>
                  <!-- </giggly-tab-content> -->
                </giggly-tab>
              </giggly-tabs>
