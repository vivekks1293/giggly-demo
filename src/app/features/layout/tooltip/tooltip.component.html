Giggly Tooltip Component

The Giggly Tooltip component is a lightweight and stylish UI element designed to provide additional context or information when users interact with elements on the page. Whether triggered by hover or click, the tooltip enhances usability without cluttering the UI.

Preview

Hover me Click me

How to Use

              
                <giggly-tooltip content="This is a tooltip!" position="top" trigger="hover">
                <giggly-button variant="primary">Hover me</giggly-button>
              </giggly-tooltip>
          
              <giggly-tooltip content="Click for info!" position="right" trigger="click">
                <giggly-button variant="secondary">Click me</giggly-button>
              </giggly-tooltip>
