Giggly Draggable-Window Component

The Giggly Draggable Window is a flexible, fun, and interactive floating panel that users can drag around the screen. It's designed to provide a smooth, modern feel while keeping the interface playful and professional.

Window Preview Preview

This is a resizable (by input) draggable window!

How to Use

              
                <giggly-draggable-window 
                [title]="'Feedback Box'"
                [width]="'400px'"
                [height]="'300px'"
                (closed)="showWindow = false"
                *ngIf="showWindow">
                <p>This is a resizable (by input) draggable window!</p>
              </giggly-draggable-window>
