Giggly Accordion Component

The Giggly Accordion component is a versatile UI element that provides expandable and collapsible sections for organizing content in a structured yet playful manner. This component is perfect for FAQs, settings panels, or any scenario where content needs to be displayed in a progressive disclosure pattern.

How to Use

              
                <giggly-accordion title="First Section" [isExpanded]="true">
                <p>This is the content of the first section.</p>
              </giggly-accordion>
          
              <giggly-accordion title="Second Section">
                <p>This is the content of the second section.</p>
              </giggly-accordion>
          
              <giggly-accordion title="Third Section" [isExpanded]="false">
                <p>This is the content of the third section, initially collapsed.</p>
              </giggly-accordion>
