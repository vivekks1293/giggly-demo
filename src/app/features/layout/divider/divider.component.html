Giggly Divider Component

The Giggly Divider component is designed to provide visual separation between sections of content in both horizontal and vertical orientations. It aligns with the playful yet professional theme of the Giggly UI library.

How to Use

              
                <h2>Section 1</h2>
                <giggly-divider orientation="horizontal" color="lightgray" thickness="2px"></giggly-divider>
                <h2>Section 2</h2>
            
                <div style="display: flex; align-items: center;">
                    <span>Item 1</span>
                    <giggly-divider orientation="vertical" color="darkgray" thickness="3px"></giggly-divider>
                    <span>Item 2</span>
                </div>
