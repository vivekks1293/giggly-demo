Giggly Loaders Component

The Giggly Loader is a stylish, animated indicator used to represent loading states in your application. It brings charm and interactivity to otherwise static wait times, keeping the user experience engaging and on-brand with the Giggly theme.

Preview

How to Use

              
                <giggly-loader type="dots"></giggly-loader>
                <giggly-loader type="ring" size="50px" color="blue"></giggly-loader>
                <giggly-loader type="bubbles" size="40px"></giggly-loader>
