Giggly Voting Component

The Giggly Votes library provides a flexible, animated voting component for Angular applications that follows the same design principles as your Giggly UI system.

How to Use

              
                <giggly-voting 
                [question]="question"
                [options]="options"
                [showTimeLeft]="true"
                [timeLeftText]="timeLeftText"
                (voteSelected)="handleVote($event)">
              </giggly-voting>
