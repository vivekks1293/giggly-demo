Giggly Text Area

The Giggly Textarea is a multi-line input component designed for capturing longer text inputs in a playful yet professional manner. It is part of the Giggly UI library, offering a seamless user experience with auto-resizing capabilities and easy integration with Angular applications.

How to Use

              
                <giggly-textarea 
                label="Comments" 
                placeholder="Write your thoughts..." 
                [rows]="3" 
                [maxRows]="6"
                [required]="true" 
                errorMessage="Comments are required"
                (valueChange)="handleTextareaChange($event)">
            </giggly-textarea>
            
            <giggly-textarea 
                label="Feedback" 
                placeholder="Your feedback here..." 
                [disabled]="true">
            </giggly-textarea>
