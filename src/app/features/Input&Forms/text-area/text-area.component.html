Giggly Text Area

The Giggly Button component provides a set of reusable buttons with various variants and states. These buttons are designed to be playful, interactive, and user-friendly.

How to Use

              
                <giggly-textarea 
                label="Comments" 
                placeholder="Write your thoughts..." 
                [rows]="3" 
                [maxRows]="6"
                [required]="true" 
                errorMessage="Comments are required"
                (valueChange)="handleTextareaChange($event)">
            </giggly-textarea>
            
            <giggly-textarea 
                label="Feedback" 
                placeholder="Your feedback here..." 
                [disabled]="true">
            </giggly-textarea>
