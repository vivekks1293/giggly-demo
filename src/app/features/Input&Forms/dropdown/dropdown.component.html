Giggly Dropdown

The Giggly Select/Dropdown component is a versatile and playful UI element that allows users to choose from a list of options in both single and multi-select modes. Designed to match the vibrant and professional theme of the Giggly UI library, it offers flexibility and an engaging user experience with powerful features.

How to Use

              
                <giggly-select 
  [options]="['Apple', 'Banana', 'Orange']" 
  placeholder="Select a fruit"
  [(ngModel)]="selectedFruit" 
  (selectionChange)="handleSelectionChange($event)"
  (searchResults)="handleSearchResults($event)">
</giggly-select>

<h3>Multi-Select with Objects</h3>

<giggly-select 
  [options]="employeeList" 
  displayField="EmpName" 
  multiple="true" 
  searchEnabled="true"
  [(ngModel)]="selectedEmployees" 
  (selectionChange)="handleSelectionChange($event)"
  (searchResults)="handleSearchResults($event)">
</giggly-select>
