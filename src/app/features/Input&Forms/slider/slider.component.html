Giggly Slider

The Giggly Slider is a stylish and interactive UI component that allows users to select a numeric value within a defined range. Designed to align with the fun yet professional theme of the Giggly UI library, it offers smooth interactions and flexible customization for seamless integration into Angular applications.

How to Use

              
                <giggly-slider 
                label="Volume Control" 
                [min]="0" 
                [max]="100" 
                [(ngModel)]="volume" 
                (valueChange)="handleSliderChange($event)">
              </giggly-slider>
            
              <giggly-slider 
                label="Brightness Level" 
                [min]="0" 
                [max]="200" 
                [disabled]="true" 
                [(ngModel)]="brightness">
              </giggly-slider>
