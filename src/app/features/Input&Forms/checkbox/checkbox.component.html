Giggly Checkbox

The Giggly Checkbox is a customizable and playful UI component designed to enhance user interaction with checkboxes. It aligns with the fun yet professional theme of the Giggly UI library, providing a stylish and accessible solution for form inputs.

Checkbox Example

Checkbox Status: {{ isAccepted ? 'Accepted' : 'Not Accepted' }}

How to Use

              
                <giggly-checkbox 
                label="Accept Terms and Conditions" 
                [(ngModel)]="isAccepted"
                (checkedChange)="handleCheckboxChange($event)">
            </giggly-checkbox>
            
            <giggly-checkbox 
                label="Subscribe to Newsletter" 
                [disabled]="true">
            </giggly-checkbox>
            
            <p>Checkbox Status: {{ isAccepted ? 'Accepted' : 'Not Accepted' }}</p>
