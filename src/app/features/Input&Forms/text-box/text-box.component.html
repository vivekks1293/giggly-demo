Giggly Text Box

The Giggly Textarea is a multi-line input component designed for capturing longer text inputs in a playful yet professional manner. It is part of the Giggly UI library, offering a seamless user experience with auto-resizing capabilities and easy integration with Angular applications.

Text Input Examples

How to Use

              
                <giggly-text-box 
                label="Username" 
                placeholder="Enter your username" 
                [required]="true"
                [(ngModel)]="username" 
                errorMessage="Username is required"
                (valueChange)="handleValueChange($event)">
              </giggly-text-box>
              
              <giggly-text-box 
                label="Email" 
                placeholder="Enter your email" 
                type="email" 
                [required]="true"
                errorMessage="Email is required" 
                (valueChange)="handleValueChange($event)">
              </giggly-text-box>
              
              <giggly-text-box 
                label="Password" 
                placeholder="Enter your password" 
                type="password" 
                [required]="true"
                errorMessage="Password is required" 
                [disabled]="false">
              </giggly-text-box>
