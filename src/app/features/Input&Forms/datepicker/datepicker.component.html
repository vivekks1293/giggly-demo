Giggly Date Picker

The Giggly Date Picker is a modern and playful calendar-based date selection component that provides a fun yet professional experience. Designed with user-friendly interactions in mind, it allows for flexible date selection while maintaining consistency with the Giggly UI theme.

Button Preview

Selected Date: {{ selectedDate | date }}

How to Use

              
                <giggly-date-picker 
                label="Select Your Birthday" 
                [minDate]="'2000-01-01'" 
                [maxDate]="'2030-12-31'" 
                [(ngModel)]="selectedDate"
                (dateChange)="handleDateChange($event)">
              </giggly-date-picker>
              
              <giggly-date-picker 
                label="Event Date" 
                [(ngModel)]="eventDate" 
                [disabled]="true">
              </giggly-date-picker>
              
              <p>Selected Date: { { selectedDate }}</p>
              <p>Event Date: { { eventDate }}</p>
