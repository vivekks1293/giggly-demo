Giggly File Upload

The Giggly File Upload component is a versatile file uploader that provides both drag-and-drop and button-based upload functionalities. It allows users to select single or multiple files, supports various file formats, and emits an event containing the uploaded files for further processing.

How to Use

              
                <giggly-file-upload 
                label="Upload Your Documents" 
                [multiple]="true"
                accept="image/*,application/pdf"
                (fileSelected)="handleFileUpload($event)">
              </giggly-file-upload>
          
              <p *ngIf="uploadedFiles.length > 0">
                Uploaded Files: 
                <ul>
                  <li *ngFor="let file of uploadedFiles"></li>
                </ul>
              </p>
