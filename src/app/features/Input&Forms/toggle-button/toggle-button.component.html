Giggly Switch

The Giggly Toggle Switch is a stylish and playful component that allows users to toggle between two states, such as ON/OFF or Enabled/Disabled. Designed to fit the fun and professional theme of the Giggly UI library, it offers an engaging and highly customizable user experience with seamless integration into Angular applications.

Toggle Switch Example

Notifications: {{ notificationsEnabled ? 'Enabled' : 'Disabled' }}

Dark Mode: {{ darkMode ? 'Enabled' : 'Disabled' }}

How to Use

              
                <giggly-toggle 
                label="Enable Notifications" 
                [(ngModel)]="notificationsEnabled"
                (toggleChange)="handleToggleChange($event)">
              </giggly-toggle>
              
              <giggly-toggle 
                label="Dark Mode" 
                [(ngModel)]="darkMode" 
                [disabled]="true">
              </giggly-toggle>
              
              <p>Notifications: {{ notificationsEnabled ? 'Enabled' : 'Disabled' }}</p>
              <p>Dark Mode: {{ darkMode ? 'Enabled' : 'Disabled' }}</p>
