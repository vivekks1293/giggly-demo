Giggly Radio Button

The Giggly Radio Button component is a customizable and playful solution for selecting a single option from a group. Designed to align with the fun yet professional theme of the Giggly UI library, it offers an engaging user experience with flexible styling and dynamic functionality.

Button Preview

Selected Color: {{ selectedColor }}

How to Use

              
                <giggly-radio-group 
                [options]="[
                { label: 'Red', value: 'red' },
                { label: 'Blue', value: 'blue' },
                { label: 'Green', value: 'green' }
                ]" 
                [(ngModel)]="selectedColor" 
                (valueChange)="onColorChange($event)" 
                direction="horizontal">
              </giggly-radio-group>
              
              <p>Selected Color: {{ selectedColor }}</p>
