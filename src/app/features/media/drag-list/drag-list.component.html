Giggly Drag-List Component

The Giggly Drag List component is a playful yet professional feature that allows users to reorder items through a simple drag-and-drop interface. Built on Angular's CDK drag-and-drop capabilities, it integrates seamlessly into your application with minimal setup.

Preview

How to Use

              
                <giggly-drag-list [items]="items" (itemsChange)="onReorder($event)"></giggly-drag-list>
