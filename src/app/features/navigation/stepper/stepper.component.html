Giggly Stepper Component

The Giggly Stepper is a fun and intuitive multi-step progress indicator designed to guide users through a sequential process. It visually represents each step in a process, making it easy for users to track their progress.

How to Use

              
                <giggly-stepper [steps]="steps" [(activeStep)]="currentStep"
                (stepChange)="handleStepChange($event)">
            </giggly-stepper>
        
            <giggly-button (click)="prevStep()" variant="primary">Previous</giggly-button>
            <giggly-button (click)="nextStep()" variant="secondary">Next</giggly-button>
