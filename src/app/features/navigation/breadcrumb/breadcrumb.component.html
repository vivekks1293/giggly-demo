Giggly Breadcrumb Component

The Giggly Breadcrumb component provides a structured way to display navigation hierarchy, improving usability and enhancing navigation within an application. With a fun yet professional design, it aligns perfectly with the Giggly UI theme.

Preview

How to Use

              
                <giggly-breadcrumb [breadcrumbs]="breadcrumbs" (breadcrumbClick)="handleBreadcrumbClick($event)">
                </giggly-breadcrumb>
