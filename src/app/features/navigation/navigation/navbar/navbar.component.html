Giggly Navbar Component

The Giggly Navbar is a versatile, multi-level navigation component designed for modern web applications. It provides N-depth menu support, allowing users to navigate complex structures effortlessly.

Preview

How to Use

              
                <giggly-navbar [menuItems]="menuList" [sticky]="true" themeColor="primary"
                (onMenuSelect)="handleMenuClick($event)">
            </giggly-navbar>
