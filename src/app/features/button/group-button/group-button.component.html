Giggly Button Group

The Group Button is a versatile UI component that allows grouping multiple buttons together, providing a cohesive design and functionality. It is part of the Giggly UI library, designed to add playful yet professional elements to your user interface.

How to Use

Text only

              
                <giggly-button-group
                [buttons]="[
                { icon: 'fas fa-bold', variant: 'primary' } ,
                { icon: 'fas fa-italic', variant: 'secondary' } ,
                {   icon: 'fas fa-underline', variant: 'danger' } 
                ]"
                orientation="horizontal"
              >

Text only

    
<giggly-button-group
  [buttons]="[
  { label: 'Bold', variant: 'primary' },
  { label: 'Italic', variant: 'secondary' },
  { label: 'Underline', variant: 'danger' }
  ]"
  orientation="horizontal"
>
</giggly-button-group>

Icons + Text

    
<giggly-button-group
  [buttons]="[

  ]"
  orientation="horizontal"
>
</giggly-button-group>
